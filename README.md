# xterm-recorder

A terminal session recording tool that will not only record terminal output but also audio.

## Install

```bash
$ npm i -g xterm-recorder
```

## Usage

```bash
$ xterm-recorder rec  # this will generate out.cast, out.mp3, evt.json, audio.pcm on $PWD
```
